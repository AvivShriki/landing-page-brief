# שאלון אפיון לדף נחיתה — אביב דיגיטל

עמוד שאלון מקצועי שנשלח ללקוחות חדשים לפני בניית דף נחיתה.
הלקוח ממלא את הפרטים, ובלחיצת כפתור נשלח סיכום מסודר של כל התשובות
לוואטסאפ של אביב. בסוף העמוד יש כפתור נפרד לשליחת קבצים (לוגו, תמונות, סרטונים).

## מה יש בעמוד
- 8 פרקי שאלות: פרטים אישיים, על העסק, השירות והמטרה, קהל היעד, בידול ואמון, סגנון ועיצוב, תוכן ונכסים, טכני ולו"ז.
- שמירה אוטומטית של התשובות בדפדפן — אפשר לעצור ולחזור.
- פס התקדמות, ולידציה לשדות חובה, וכפתור העתקת הסיכום ללוח.
- עברית מלאה, מותאם למובייל (RTL).

## עיצוב
כיוון עיצובי שנבחר בעזרת קנבה: בסיס דיו כהה, מבטא זהב־טרקוטה, רקע שמנת.
טיפוגרפיה: Frank Ruhl Libre לכותרות, Assistant לטקסט רץ.

## מבנה
- `index.html` — העמוד
- `css/style.css` — עיצוב
- `js/form.js` — צ'יפים, שמירה, ולידציה ושליחה לוואטסאפ
